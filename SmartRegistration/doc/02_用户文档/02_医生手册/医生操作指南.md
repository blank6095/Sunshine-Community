# 医生操作指南

## 1. 概述

本文档是智能挂号预约系统的医生操作指南，旨在帮助医生了解和使用系统的各项功能。

## 2. 登录系统

### 2.1 访问地址

- **系统地址**：http://localhost:8080/api/v1
- **登录接口**：`POST /auth/login`

### 2.2 登录凭证

- **账号**：由管理员创建
- **密码**：初始密码由管理员设置

### 2.3 登录流程

1. 发送登录请求：
   ```json
   {
     "username": "doctor1",
     "password": "password"
   }
   ```

2. 系统返回JWT令牌和用户信息：
   ```json
   {
     "code": 200,
     "message": "成功",
     "data": {
       "token": "eyJhbGciOiJIUzUxMiJ9...",
       "user": {
         "id": 2,
         "username": "doctor1",
         "name": "张医生",
         "role": "DOCTOR",
         "status": "ACTIVE"
       }
     },
     "error": null
   }
   ```

3. 在后续请求的请求头中添加：
   ```
   Authorization: Bearer {token}
   ```

## 3. 医生功能

### 3.1 个人信息管理

#### 3.1.1 获取个人信息

- **接口**：`GET /users/me`
- **响应**：当前用户信息

#### 3.1.2 更新个人信息

- **接口**：`PUT /users/me`
- **请求体**：
  ```json
  {
    "name": "张医生",
    "phone": "13800138000",
    "email": "doctor1@example.com"
  }
  ```

#### 3.1.3 修改密码

- **接口**：`PUT /users/me/password`
- **请求体**：
  ```json
  {
    "oldPassword": "password",
    "newPassword": "newpassword123"
  }
  ```

### 3.2 预约管理

#### 3.2.1 获取我的预约列表

- **接口**：`GET /appointments/doctor/me`
- **参数**：
  - `status`：状态（可选）
  - `startDate`：开始日期（可选）
  - `endDate`：结束日期（可选）
- **响应**：我的预约列表

#### 3.2.2 获取预约详情

- **接口**：`GET /appointments/{id}`
- **响应**：预约详情

#### 3.2.3 更新预约状态

- **接口**：`PUT /appointments/{id}`
- **请求体**：
  ```json
  {
    "status": "COMPLETED"
  }
  ```

#### 3.2.4 取消预约

- **接口**：`PUT /appointments/{id}/cancel`

### 3.3 排班管理

#### 3.3.1 获取我的排班列表

- **接口**：`GET /schedules/doctor/me`
- **参数**：
  - `date`：日期（可选，格式：yyyy-MM-dd）
- **响应**：我的排班列表

#### 3.3.2 创建排班

- **接口**：`POST /schedules`
- **请求体**：
  ```json
  {
    "doctorId": 1,
    "departmentId": 1,
    "date": "2026-05-01",
    "startTime": "08:00",
    "endTime": "12:00",
    "totalSlots": 20,
    "availableSlots": 20,
    "status": "ACTIVE"
  }
  ```

#### 3.3.3 更新排班

- **接口**：`PUT /schedules/{id}`
- **请求体**：
  ```json
  {
    "totalSlots": 25,
    "availableSlots": 25,
    "status": "ACTIVE"
  }
  ```

#### 3.3.4 删除排班

- **接口**：`DELETE /schedules/{id}`

### 3.4 患者管理

#### 3.4.1 获取患者信息

- **接口**：`GET /users/{id}`
- **响应**：患者信息

#### 3.4.2 查看患者预约历史

- **接口**：`GET /appointments/patient/{patientId}`
- **响应**：患者的预约历史

## 4. 操作流程

### 4.1 日常工作流程

1. **登录系统**：使用账号密码登录
2. **查看预约**：查看当天的预约列表
3. **处理预约**：按顺序处理患者预约
4. **更新状态**：完成预约后更新状态
5. **管理排班**：创建或更新未来的排班

### 4.2 排班管理流程

1. **查看现有排班**：了解当前排班情况
2. **创建新排班**：为未来日期创建排班
3. **调整排班**：根据实际情况调整排班
4. **取消排班**：如需取消，提前通知患者

### 4.3 预约处理流程

1. **接收预约**：系统自动接收患者预约
2. **确认预约**：确认患者信息和预约时间
3. **准备接诊**：根据患者症状准备
4. **处理预约**：按预约时间接诊
5. **完成预约**：标记预约为已完成
6. **记录信息**：记录诊疗信息

## 5. 常见问题

### 5.1 登录问题

- **问题**：登录失败
  **解决**：检查用户名和密码是否正确，确保用户状态为ACTIVE

- **问题**：令牌过期
  **解决**：使用刷新令牌接口获取新令牌

### 5.2 预约问题

- **问题**：预约冲突
  **解决**：检查排班时间，避免时间冲突

- **问题**：预约状态更新失败
  **解决**：检查网络连接，确保权限正确

### 5.3 排班问题

- **问题**：排班创建失败
  **解决**：检查参数是否正确，确保日期格式正确

- **问题**：排班时间冲突
  **解决**：检查现有排班，避免时间重叠

# 预约流程指南

## 1. 概述

本文档详细说明智能挂号预约系统的预约流程，帮助患者了解如何使用系统进行挂号预约。

## 2. 预约前准备

### 2.1 注册账号

- **注册接口**：`POST /auth/register`
- **请求体**：
  ```json

    "password": "password",
    "name": "王患者",
    "gender": "男",
    "age": 30,
    "phone": "13800138000",
    "email": "patient1@example.com",
    "role": "PATIENT",
    "status": "ACTIVE"
  }
  ```

## 4. 预约管理

### 4.1 查看我的预约

- **接口**：`GET /appointments/patient/me`
- **响应**：我的预约列表

### 4.2 取消预约

- **接口**：`PUT /appointments/{id}/cancel`
- **响应**：
  ```json
  {
    "code": 200,
    "message": "成功",
    "data": {
      "id": 1,
      "status": "CANCELLED"
    },
    "error": null
  }
  ```

### 4.3 预约状态

| 状态 | 描述 |
|------|------|
| PENDING | 待确认 |
| CONFIRMED | 已确认 |
| COMPLETED | 已完成 |
| CANCELLED | 已取消 |

## 5. 就诊流程

### 5.1 预约确认

- 系统会发送预约确认通知
- 确认预约时间和地点
- 准备相关资料

### 5.2 就诊当天

- 提前30分钟到达医院
- 凭预约号到指定科室报到
- 等待叫号就诊

### 5.3 就诊后

- 按医生建议进行治疗
- 可查看预约记录和医嘱
- 如需复诊，可再次预约

## 6. 常见问题

### 6.1 预约失败

- **问题**：预约提交失败
  **解决**：检查网络连接，确保号源充足

### 6.2 预约取消

- **问题**：需要取消预约
  **解决**：在系统中取消预约，或联系医院

### 6.3 预约时间调整

- **问题**：需要调整预约时间
  **解决**：取消原预约，重新预约其他时间

### 6.4 忘记密码

- **问题**：忘记登录密码
  **解决**：使用密码重置功能，或联系客服

## 7. 预约须知

- 请至少提前24小时预约
- 如需取消预约，请至少提前4小时
- 就诊时请携带身份证和医保卡
- 如实描述症状，以便医生准备
- 遵守医院的就诊秩序

## 8. 系统功能说明

### 8.1 预约提醒

- 系统会在预约前发送提醒
- 可在个人设置中配置提醒方式

### 8.2 就诊记录

- 可查看历史就诊记录
- 可查看医生的诊断和建议
- 可下载就诊报告

### 8.3 在线咨询

- 部分医生支持在线咨询
- 可通过系统发送咨询消息
- 医生会在工作时间回复

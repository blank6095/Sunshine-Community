# 常见问题

## 1. 概述

本文档收集了智能挂号预约系统的常见问题和解决方案，帮助用户快速解决使用过程中遇到的问题。

## 2. 登录问题

### 2.1 无法登录

- **问题**：输入正确的用户名和密码但无法登录
- **可能原因**：
  - 用户名或密码错误
  - 用户账号被锁定
  - 用户账号状态为非活跃
  - 系统故障
- **解决方案**：
  - 检查用户名和密码是否正确
  - 联系管理员解锁账号
  - 联系管理员激活账号
  - 检查系统状态

### 2.2 忘记密码

- **问题**：忘记登录密码
- **解决方案**：
  - 使用密码重置功能：`POST /auth/forgot-password`
  - 联系管理员重置密码

### 2.3 账号锁定

- **问题**：连续登录失败后账号被锁定
- **解决方案**：
  - 等待30分钟后自动解锁
  - 联系管理员手动解锁

## 3. 预约问题

### 3.1 无法预约

- **问题**：提交预约请求失败
- **可能原因**：
  - 号源已用完
  - 排班已取消
  - 网络连接问题
  - 系统故障
- **解决方案**：
  - 选择其他排班
  - 检查排班状态
  - 检查网络连接
  - 联系客服

### 3.2 预约取消

- **问题**：需要取消已提交的预约
- **解决方案**：
  - 在系统中取消预约：`PUT /appointments/{id}/cancel`
  - 联系医院取消

### 3.3 预约时间调整

- **问题**：需要调整预约时间
- **解决方案**：
  - 取消原预约后重新预约
  - 联系医院调整

### 3.4 预约确认

- **问题**：提交预约后未收到确认
- **可能原因**：
  - 系统延迟
  - 邮箱或短信未收到
  - 预约未成功提交
- **解决方案**：
  - 查看预约列表确认状态
  - 检查垃圾邮件
  - 重新提交预约

## 11. 故障排查

### 11.1 基本排查步骤

1. **检查网络连接**：确保网络连接正常
2. **检查系统状态**：查看系统健康检查端点
3. **检查错误信息**：记录并分析错误信息
4. **尝试重新操作**：可能是临时故障
5. **联系支持**：如果问题持续存在

### 11.2 高级排查

- **查看日志**：检查系统日志获取详细信息
- **测试API**：使用工具测试API接口
- **检查配置**：验证系统配置是否正确
- **性能监控**：检查系统性能指标

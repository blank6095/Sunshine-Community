# 账户管理指南

## 1. 概述

本文档详细说明智能挂号预约系统的账户管理功能，帮助患者了解如何注册、登录和管理个人账户。

## 2. 账户注册

### 2.1 注册流程

1. **访问注册接口**：`POST /auth/register`
2. **提交注册信息**：
   ```json
   {
     "username": "patient1",
     "password": "password",
     "name": "王患者",
     "gender": "男",
     "age": 30,
     "phone": "13800138000",
     "email": "patient1@example.com",
     "role": "PATIENT",
     "status": "ACTIVE"
   }
   ```
3. **接收注册成功通知**
4. **登录系统**

### 2.2 注册注意事项

- 用户名必须唯一
- 密码长度至少8位
- 必须提供真实的个人信息
- 确保联系方式正确无误

## 3. 账户登录

### 3.1 登录流程

1. **访问登录接口**：`POST /auth/login`
2. **提交登录凭证**：
   ```json
   {
     "username": "patient1",
     "password": "password"
   }
   ```
3. **接收登录成功响应**：
   ```json
   {
     "code": 200,
     "message": "成功",
     "data": {
       "token": "eyJhbGciOiJIUzUxMiJ9...",
       "user": {
         "id": 3,
         "username": "patient1",
         "name": "王患者",
         "role": "PATIENT",
         "status": "ACTIVE"
       }
     },
     "error": null
   }
   ```
4. **保存令牌**：在后续请求中使用

### 3.2 登录注意事项

- 确保用户名和密码正确
- 注意区分大小写
- 连续登录失败会被暂时锁定

## 4. 个人信息管理

### 4.1 查看个人信息

- **接口**：`GET /users/me`
- **响应**：
  ```json
  {
    "code": 200,
    "message": "成功",
    "data": {
      "id": 3,
      "username": "patient1",
      "name": "王患者",
      "gender": "男",
      "age": 30,
      "phone": "13800138000",
      "email": "patient1@example.com",
      "role": "PATIENT",
      "status": "ACTIVE"
    },
    "error": null
  }
  ```

### 4.2 更新个人信息

- **接口**：`PUT /users/me`
- **请求体**：
  ```json
  {
    "name": "王患者",
    "phone": "13900139000",
    "email": "patient1@example.com"
  }
  ```
- **响应**：更新后的个人信息

### 4.3 修改密码

- **接口**：`PUT /users/me/password`
- **请求体**：
  ```json
  {
    "oldPassword": "password",
    "newPassword": "newpassword123"
  }
  ```
- **响应**：
  ```json
  {
    "code": 200,
    "message": "密码修改成功",
    "data": null,
    "error": null
  }
  ```

## 5. 安全设置

### 5.1 密码安全

- 定期更换密码
- 使用强密码（包含字母、数字和特殊字符）
- 不要与其他网站使用相同的密码
- 不要将密码告知他人

### 5.2 账户安全

- 启用双重认证（如果支持）
- 定期检查账户活动
- 及时更新联系方式
- 发现异常活动立即联系客服

### 5.3 隐私设置

- 控制个人信息的可见性
- 设置消息通知偏好
- 管理数据共享设置

## 6. 账户问题

### 6.1 忘记密码

- **接口**：`POST /auth/forgot-password`
- **请求体**：
  ```json
  {
    "email": "patient1@example.com"
  }
  ```
- **响应**：系统会发送密码重置链接到邮箱

### 6.2 账户锁定

- **原因**：连续登录失败多次
- **解决**：等待30分钟后重新尝试，或联系客服解锁

### 6.3 账户注销

- **接口**：`DELETE /users/me`
- **响应**：
  ```json
  {
    "code": 200,
    "message": "账户注销成功",
    "data": null,
    "error": null
  }
  ```

## 7. 常见问题

### 7.1 注册失败

- **问题**：注册时提示用户名已存在
  **解决**：选择其他用户名

- **问题**：注册信息验证失败
  **解决**：检查信息格式是否正确

### 7.2 登录问题

- **问题**：登录失败
  **解决**：检查用户名和密码是否正确

- **问题**：令牌过期
  **解决**：重新登录获取新令牌

### 7.3 信息更新

- **问题**：无法更新个人信息
  **解决**：检查信息格式是否正确，确保权限充足

- **问题**：密码修改失败
  **解决**：检查旧密码是否正确，新密码是否符合要求

## 8. 账户管理最佳实践

- **定期检查**：定期检查个人信息和账户活动
- **安全使用**：在安全的环境中使用账户
- **及时更新**：及时更新个人信息和联系方式
- **妥善保管**：妥善保管登录凭证

# 智慧医院挂号预约系统 - 开发环境搭建指南

## 1 环境要求

### 1.1 开发工具

| 工具 | 版本要求 | 说明 |
|------|----------|------|
| Node.js | >= 18.0.0 | 推荐使用LTS版本 |
| pnpm | >= 8.0.0 | 包管理器 |
| HBuilderX | >= 3.8.0 | UniApp推荐IDE |
| VS Code | 最新版 | 可选，配合插件使用 |

### 1.2 环境检查

在开始搭建前，请先检查本地环境：

```bash
# 检查Node.js版本
node -v

# 检查pnpm版本
pnpm -v

# 检查npm版本
npm -v
```

如果版本不满足要求，请先升级或安装对应版本。

## 2 项目初始化

### 2.1 克隆项目

```bash
# 克隆项目代码
git clone <repository-url>

# 进入项目目录
cd client
```

### 2.2 安装依赖

```bash
# 使用pnpm安装依赖
pnpm install
```

### 2.3 安装HBuilderX插件

1. 打开HBuilderX
2. 进入 `工具` -> `插件安装`
3. 安装以下插件：
   - Sass/SCSS编译
   - ESLint
   - Prettier

## 3 开发配置

### 3.1 环境变量配置

在项目根目录创建 `.env.development` 文件：

```bash
# 开发环境API地址
VITE_API_BASE_URL=http://localhost:8080/api/v1
```

在项目根目录创建 `.env.production` 文件：

```bash
# 生产环境API地址
VITE_API_BASE_URL=https://api.example.com/api/v1
```

### 3.2 微信小程序配置

如果需要开发微信小程序版本，需要在 `manifest.json` 中配置：

```json
{
  "mp-weixin": {
    "appid": "your-weixin-appid",
    "setting": {
      "urlCheck": false,
      "es6": true,
      "postcss": true,
      "minified": true
    },
    "usingComponents": true
  }
}
```

### 3.3 H5配置

在 `manifest.json` 中配置H5路由模式：

```json
{
  "h5": {
    "router": {
      "mode": "history",
      "base": "/"
    },
    "devServer": {
      "port": 8080,
      "https": false
    }
  }
}
```

## 4 开发命令

### 4.1 常用命令

| 命令 | 说明 | 适用平台 |
|------|------|----------|
| pnpm dev:h5 | 开发模式启动H5 | H5 |
| pnpm dev:mp-weixin | 开发模式启动微信小程序 | 微信小程序 |
| pnpm dev:app | 开发模式启动App | Android/iOS |
| pnpm build:h5 | 生产构建H5 | H5 |
| pnpm build:mp-weixin | 生产构建微信小程序 | 微信小程序 |
| pnpm build:app | 生产构建App | Android/iOS |

### 4.2 开发H5版本

```bash
# 启动H5开发服务器
pnpm dev:h5
```

启动成功后，访问 `http://localhost:8080`

### 4.3 开发微信小程序版本

```bash
# 启动微信小程序开发模式
pnpm dev:mp-weixin
```

构建完成后：
1. 打开微信开发者工具
2. 导入项目 `dist/dev/mp-weixin` 目录
3. 开始调试

### 4.4 开发App版本

```bash
# 启动App开发模式
pnpm dev:app
```

启动成功后，使用HBuilderX连接手机进行真机调试。

## 5 API接口说明

### 5.1 后端API地址

| 环境 | 地址 |
|------|------|
| 开发环境 | http://localhost:8080/api/v1 |
| 生产环境 | https://api.example.com/api/v1 |

### 5.2 接口认证

系统使用JWT进行身份认证，登录后返回的token需要存储并在后续请求中携带。

```typescript
// 请求头格式
Authorization: Bearer {token}
```

### 5.3 主要接口模块

| 模块 | 接口数 | 说明 |
|------|--------|------|
| 认证模块 | 3 | 登录、注册、刷新令牌 |
| 用户模块 | 5 | 用户CRUD |
| 科室模块 | 5 | 科室CRUD |
| 医生模块 | 7 | 医生CRUD、筛选 |
| 排班模块 | 9 | 排班CRUD、筛选 |
| 预约模块 | 11 | 预约CRUD、筛选 |

详细接口请参考 [API文档.md](./API文档.md)

## 8 常见问题

### 8.1 依赖安装失败

如果pnpm install失败，尝试：

```bash
# 清理缓存
pnpm store prune

# 删除node_modules重试
rm -rf node_modules
pnpm install
```

### 8.2 微信开发者工具导入失败

1. 确保已开启微信开发者工具的服务端口
2. 检查project.config.json中的appid是否正确
3. 确保dist/dev/mp-weixin目录已正确生成

### 8.3 H5跨域问题

开发环境如果遇到跨域问题，可以在 `vite.config.ts` 中配置代理：

```typescript
import { defineConfig } from 'vite'
import uni from '@dcloudio/vite-plugin-uni'

export default defineConfig({
  plugins: [uni()],
  server: {
    proxy: {
      '/api': {
        target: 'http://localhost:8080',
        changeOrigin: true
      }
    }
  }
})
```

### 8.4 App真机调试失败

1. 确保手机已开启开发者模式
2. 确保手机与电脑在同一网络
3. 检查HBuilderX的运行配置
4. 尝试重新编译再运行

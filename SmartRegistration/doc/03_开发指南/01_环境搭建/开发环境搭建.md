# 开发环境搭建

## 1. 概述

本文档详细说明智能挂号预约系统的开发环境搭建步骤，帮助开发人员快速搭建开发环境。

## 2. 环境要求

### 2.1 硬件要求

- **CPU**：至少4核
- **内存**：至少8GB
- **磁盘**：至少50GB可用空间

### 2.2 软件要求

| 软件 | 版本 | 用途 |
|------|------|------|
| JDK | 17+ | Java开发环境 |
| MySQL | 8.0+ | 数据库 |
| Maven | 3.6+ | 构建工具 |
| IntelliJ IDEA | 2023+ | 开发IDE |
| Git | 2.20+ | 版本控制 |

## 3. 环境搭建步骤

### 3.1 JDK安装

1. **下载JDK**：
   - 访问Oracle官网或OpenJDK官网
   - 下载JDK 17或更高版本

2. **安装JDK**：
   - 运行安装程序
   - 按照向导完成安装

3. **配置环境变量**：
   - 设置`JAVA_HOME`环境变量指向JDK安装目录
   - 将`%JAVA_HOME%\bin`添加到`PATH`环境变量

4. **验证安装**：
   ```bash
   java -version
   javac -version
   ```

### 3.2 MySQL安装

1. **下载MySQL**：
   - 访问MySQL官网
   - 下载MySQL 8.0或更高版本

2. **安装MySQL**：
   - 运行安装程序
   - 选择"Developer Default"安装类型
   - 设置root用户密码

3. **启动MySQL服务**：
   - 启动MySQL服务
   - 确保服务正常运行

4. **验证安装**：
   ```bash
   mysql -u root -p
   ```

### 3.3 Maven安装

1. **下载Maven**：
   - 访问Maven官网
   - 下载Maven 3.6或更高版本

2. **安装Maven**：
   - 解压到指定目录
   - 设置`MAVEN_HOME`环境变量
   - 将`%MAVEN_HOME%\bin`添加到`PATH`环境变量

3. **验证安装**：
   ```bash
   mvn -version
   ```

### 3.4 IntelliJ IDEA安装

1. **下载IntelliJ IDEA**：
   - 访问JetBrains官网
   - 下载IntelliJ IDEA Ultimate或Community版本

2. **安装IntelliJ IDEA**：
   - 运行安装程序
   - 按照向导完成安装

3. **安装插件**：
   - Lombok插件
   - Spring Boot插件
   - MySQL插件

### 3.5 Git安装

1. **下载Git**：
   - 访问Git官网
   - 下载Git 2.20或更高版本

2. **安装Git**：
   - 运行安装程序
   - 按照向导完成安装

3. **验证安装**：
   ```bash
   git --version
   ```

## 4. 项目克隆与配置

### 4.1 克隆项目

1. **克隆代码**：
   ```bash
   git clone https://github.com/your-repo/smart-registration.git
   cd smart-registration
   ```

### 4.2 配置数据库

1. **创建数据库**：
   ```bash
   mysql -u root -p
   CREATE DATABASE smart_registration_db;
   ```

2. **导入SQL脚本**：
   ```bash
   mysql -u root -p smart_registration_db < sql/init.sql
   mysql -u root -p smart_registration_db < sql/data.sql
   ```

### 4.3 配置应用

1. **修改配置文件**：
   - 编辑`src/main/resources/application.yml`
   - 配置数据库连接信息

2. **配置Maven**：
   - 确保Maven配置正确
   - 检查依赖版本

## 5. 项目构建与运行

### 5.1 构建项目

1. **清理项目**：
   ```bash
   mvn clean
   ```

2. **编译项目**：
   ```bash
   mvn compile
   ```

3. **打包项目**：
   ```bash
   mvn package
   ```

### 5.2 运行项目

1. **运行应用**：
   ```bash
   mvn spring-boot:run
   ```

2. **验证运行**：
   - 访问 http://localhost:8080/api/v1/health
   - 检查是否返回健康状态

## 6. 开发工具配置

### 6.1 IntelliJ IDEA配置

1. **导入项目**：
   - 打开IntelliJ IDEA
   - 选择"Open"导入项目
   - 选择项目根目录

2. **配置JDK**：
   - 打开Project Structure
   - 配置Project SDK为JDK 17

3. **配置Maven**：
   - 打开Settings
   - 配置Maven路径

4. **配置运行**：
   - 配置Spring Boot运行配置
   - 运行SmartRegistrationApplication

### 6.2 代码风格配置

1. **导入代码风格**：
   - 导入项目的代码风格配置
   - 确保代码风格一致

2. **启用Lombok**：
   - 确保Lombok插件已安装
   - 启用注解处理

## 7. 调试配置

### 7.1 本地调试

1. **配置断点**：
   - 在代码中设置断点
   - 启动调试模式

2. **查看变量**：
   - 查看变量值
   - 检查调用栈

3. **修改变量**：
   - 运行时修改变量值
   - 测试不同场景

### 7.2 远程调试

1. **配置远程调试**：
   - 添加远程调试配置
   - 设置调试端口

2. **启动远程调试**：
   - 启动应用时添加调试参数
   - 连接远程调试

## 8. 依赖管理

### 8.1 依赖更新

1. **检查依赖**：
   ```bash
   mvn dependency:tree
   ```

2. **更新依赖**：
   - 编辑pom.xml文件
   - 更新依赖版本

3. **清理依赖**：
   ```bash
   mvn dependency:purge-local-repository
   ```

### 8.2 依赖冲突

1. **检测冲突**：
   ```bash
   mvn dependency:tree -Dincludes=groupId:artifactId
   ```

2. **解决冲突**：
   - 使用`<exclusions>`排除冲突依赖
   - 统一依赖版本

## 9. 开发环境问题排查

### 9.1 常见问题

| 问题 | 可能原因 | 解决方案 |
|------|---------|----------|
| 编译失败 | 依赖问题 | 检查依赖版本 |
| 数据库连接失败 | 配置错误 | 检查数据库配置 |
| 端口被占用 | 端口冲突 | 修改端口或停止占用进程 |
| 运行时错误 | 代码问题 | 检查错误日志 |

### 9.2 日志查看

1. **应用日志**：
   - 查看控制台输出
   - 查看logs目录下的日志文件

2. **数据库日志**：
   - 查看MySQL错误日志
   - 检查SQL执行情况

## 10. 开发环境最佳实践

- **使用版本控制**：所有代码都应纳入版本控制
- **保持环境一致**：开发、测试、生产环境应保持一致
- **定期备份**：定期备份代码和数据库
- **使用容器**：考虑使用Docker进行环境隔离

开发环境搭建是项目开发的基础，正确的环境配置可以提高开发效率，减少开发过程中的问题。本文档提供了详细的环境搭建步骤，帮助开发人员快速搭建开发环境，为项目开发做好准备。
